level1.1

使用 screen 或 tmux 创建一个终端，在当中运行一个程序，并尝试将终端分离。登出终端后重新恢复这个终端。说明此方式和 nohup 的区别，需要提交图文记录。

先安装screensudo apt install screen
在终端输入screen -S test新建一个终端
运行一个程序ping baidu.com

然后Ctrl+A再按D 将会话分离在后台
输入screen -ls可以查看后台挂的会话
输入screen -r test恢复会话
Ctrl+C退出ping,输入exit退出

screen和nohup相比,简单来说，screen像一个“虚拟机房”，你可以随时进出，里面的机器（程序）都在持续工作，你还能直接操作它们；而nohup更像“按下启动按钮后离开”，任务会自己完成，但中途很难再干预，只能通过查看工作日志（nohup.out文件）来了解进度
