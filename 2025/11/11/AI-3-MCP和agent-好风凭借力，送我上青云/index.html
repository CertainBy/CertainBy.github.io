1.认识agent

在了解agent这个概念之前我们得先了解一下LLM,LLM很简单就是大语言模型,而agent则是把LLM和tool结合在一起,让一个只会说话的LLM也能和人一样所以各种工具,比如查询天气,翻阅文件等等.
那么LLM是如何理解并调用tool的呢,人们给出的方案是在prompt之前再加一个system prompt(简单粗暴).

2.MCP

MCP 中文文档
在agent这个概念出现之后,因为tools的层出不穷,每个工具的prompt也不一样,因此MCP出现了,旨在
MCP 是一个开放协议，它为应用程序向 LLM 提供上下文的方式进行了标准化
