# 1.认识vLLM
一个快速、易于使用的 LLM 推理和服务库

![](https://vllm.hyper.ai/assets/images/vllm-logo-a03d4c8da188b81474f5f502aa30c4c2.png)


# 2.安装vLLM

~~前面可能得推倒重来了~~<br>
因为liunx效率更高(~~并不是windows安装vLLM失败~~),我选择了WSL2平台,用uv管理虚拟环境安装vLLM
#### 1.安装uv

先安装pixp,然后使用pipx安装uv<br>
`python3 -m pip install --user pipx`<br>
`python3 -m pipx ensurepath     # 把 ~/.local/bin 写进 PATH`
`pipx install uv`
![](https://raw.githubusercontent.com/CertainBy/blog-imgs/master/pipx%20install%20uv.png)

#### 2.安装vLLM
[vLLM参考文档](https://vllm.hyper.ai/docs/)<br>
先创建一个虚拟环境`uv venv vllm --python 3.12 --seed`<br>
并激活`source vllm/bin/activate`<br>
然后就可以在这个环境中安装vLLM了`uv pip install vllm`
*PS:可以在命令后加入--index-url https://pypi.tuna.tsinghua.edu.cn/simple/ 设置镜像源 *
