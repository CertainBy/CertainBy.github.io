1.认识vLLM

一个快速、易于使用的 LLM 推理和服务库

2.安装vLLM

前面可能得推倒重来了

因为liunx效率更高(并不是windows安装vLLM失败),我选择了WSL2平台,用uv管理虚拟环境安装vLLM

1.安装uv

先安装pixp,然后使用pipx安装uv

python3 -m pip install --user pipx

python3 -m pipx ensurepath # 把 ~/.local/bin 写进 PATH
pipx install uv

2.安装vLLM

vLLM参考文档

先创建一个虚拟环境uv venv vllm --python 3.12 --seed

并激活source vllm/bin/activate

然后就可以在这个环境中安装vLLM了uv pip install vllm
*PS:可以在命令后加入–index-url https://pypi.tuna.tsinghua.edu.cn/simple/ 设置镜像源 *

已废弃 转用wsl平台uv安装vLLM

1.认识ollama

ollama 可以简单理解为一款软件,你在上面可以方便地部署各种模型.

1.下载并安装ollama

https://ollama.com/download

2.验证安装

在cmd中输入ollama --version
